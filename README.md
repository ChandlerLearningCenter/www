
This is a demo app that integrates the following:

    Node.Js
    Express
    Custom async application architecture
    AngularJs
    Bootstrap


Notes:

    Built to leverage ExtJs style of coding within Node.js.

    Code structured for easy deployment to Windows Azure.
    
    Server-side app architecture allows multiple Node.js modules to be attached to a single Express server. 
    
    Also exposes a RESTful api.

    Live Demo:  http://clc.azurewebsites.net/
